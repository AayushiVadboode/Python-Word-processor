📝 Python Word Processor

A simple desktop-based Word Processor application built using **Python & Tkinter**.

## 🚀 Features
- Text editing area
- File menu (New, Open, Save, Exit)
- Edit menu (Cut, Copy, Paste)
- Simple and clean UI

## 🛠️ Technologies Used
- Python
- Tkinter

## 📂 Project Structure

Word_processor/
│── main.py
│── action.py
│── README.md


## ▶️ How to Run
### 1️⃣ Clone the Repository
git clone https://github.com/AayushiVadboode/Python-Word-processor.git

2️⃣ Go to Project Folder
cd Python-Word-processor

3️⃣ Run the Application
